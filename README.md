Amazon Prime Video Subtitle Downloader
A single-file userscript that downloads every subtitle track from an Amazon Prime Video title as a ZIP — with SRT, TTML and VTT output, live progress and SDH / CC / Forced detection.

Install
Install Tampermonkey or Violentmonkey.
Open public/amazon-prime-subtitle-downloader.user.js and click Raw — your userscript manager will prompt to install.
Visit any title on primevideo.com or amazon.* and start playback.
Use the floating APVSD panel to pick languages / formats and download a ZIP.
